A platform similar to OLX where the buyer and seller can use this platform B2B Market Place. But there is no 2 personas like buyer and seller, any user can be a buyer/seller.

FEATURES
- User Registration & Login
- Multiple User roles & profile management
- A platform for the seller and buyer
- To sell Machinery, accessories, tools
- Machine rental
- Recruitment & Job posting and service repair listings
- Search & Filtering System
- User Dashboard to check the overview of all the listings he has done
- Admin Management Panel[Superuser]


IMPORTANT
- Mobile first design
